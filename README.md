### Setup Steps

1. **Verify Python version**Ensure Python 3.5 or above is installed:

   ```bash
   python --version
   ```
2. **Verify Pip version**Ensure pip 19 or above is installed:

   ```bash
   pip --version
   ```
3. **Install Behave**

   ```bash
   pip install behave
   ```
4. **Install Request**

   ```bash
   pip install requests
   ```
